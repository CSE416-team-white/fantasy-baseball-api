What's included

Everything your draft app needs

A single API for all the data your fantasy baseball app needs to make smart draft decisions.

📋

Player Roster

Full MLB player database with positions, team, league, batting/pitching splits, and bio data. Filter by team, position, or league.

💰

Auction Valuations

Dollar values calculated from 3-season stat averages weighted by your league's scoring categories. Depth chart, age, and injury multipliers applied automatically.

🏥

Injury & Depth Charts

Live injury status (IL-10, IL-15, IL-60) and real-time depth chart order synced daily from ESPN. Know who's starting before you bid.

🏆

League Management

Configure roster slots, scoring categories, budget limits, and track drafted players per team. Supports auction and snake drafts.

🎯

Draft Eligibility

Pass your team ID to see which players are draftable based on your current roster, open slots, and remaining budget.

⚡

Daily Syncs

Player rosters and depth charts sync automatically every day so your app always reflects the latest MLB transactions.

Quick start

One request away from valuations

Get ranked player dollar values for any league in a single GET request.

bash 
# Get player valuations for your league
curl "https://fantasy-baseball-api.onrender.com/api/valuations/YOUR_LEAGUE_ID" \
  -H "x-api-key: YOUR_API_KEY"

# Response (sorted by dollar value, highest first)
{
  "success": true,
  "data": {
    "leagueName": "Standard 5x5 Roto",
    "valuations": [
      {
        "name": "Shohei Ohtani",
        "team": "LAD",
        "dollarValue": 38.50,
        "baseValue": 17.11,
        "draftable": true,
        "multipliers": {
          "depthChart": 1.5,
          "age":       1.0,
          "injury":    1.0
        }
      }
    ],
    "pagination": { "page": 1, "limit": 50, "total": 1323 }
  }
}
